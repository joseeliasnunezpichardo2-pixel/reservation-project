# Reservation System

Sistema de reservas desarrollado con Spring Boot y Angular.

## Tecnologías

Backend:
- Java
- Spring Boot
- Maven

Frontend:
- Angular
- TypeScript
- Node.js

## Estructura del proyecto

reservation-project
│
├── Reservation-Backend
│
└── Reservation-Frontend

## Ejecutar el backend

cd Reservation-Backend
./mvnw spring-boot:run

## Ejecutar el frontend

cd Reservation-Frontend
npm install
npm start
